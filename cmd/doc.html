Zephyr API Documentation

POST /admin/users

Create a new user (admin or waiter).

Request Body:

{ "username": "string", "password": "string", "role": "admin | waiter" }

Response:

Success: "Successfully added a new user with role = {role}"

Error: "a property for registration is missing"

DELETE /admin/users/{id}

Remove a user by their ID.

URL Parameters:

id: integer (user ID)

Response:

Success: "User successfully deleted"

Error: "User not found"

GET /admin/waiters

Get a list of all waiters.

Response:

[ { "id": 1, "username": "waiter1", "role": "waiter" }, { "id": 2, "username": "waiter2", "role": "waiter" } ]

GET /admin/waiters/{id}

Get a specific user by ID.

URL Parameters:

id: integer (user ID)

Response:

Success: user details as JSON

POST /owner/categories/{parentId}/{name}

Create a new category under a parent category.

URL Parameters:

parentId: integer (ID of the parent category)
name: string (name of the new category)

Response:

Success: "Successfully added a new category"

Error: "specify a name for category"

PATCH /owner/categories/{id}/{newName}

Update the name of a category by ID.

URL Parameters:

id: integer (category ID)
newName: string (new category name)

Response:

Success: "Successfully updated the category"

DELETE /owner/categories/{id}

Remove a category by ID.

URL Parameters:

id: integer (category ID)

Response:

Success: "Successfully removed the category"

GET /owner/categories/{parentId}

Retrieve categories for a given parent ID.

URL Parameters:

parentId: integer (parent category ID)

Response:

Success: JSON array of categories

POST /owner/products/{name}/{unit}

Add a new product with specified name and unit.

URL Parameters:

name: string (name of the product)
unit: string (unit of measurement)

Response:

Success: "Product added"

PATCH /owner/products/{id}/{newName}

Update the product name by its ID.

URL Parameters:

id: integer (product ID)
newName: string (new product name)

Response:

Success: "Product updated"

DELETE /owner/products/{id}

Remove a product by ID.

URL Parameters:

id: integer (product ID)

Response:

Success: "Product removed"

GET /owner/products/

Get a list of all products.

Response:

[ { "id": 1, "name": "Product 1", "unit": "kg", "inStock": 100 }, { "id": 2, "name": "Product 2", "unit": "liter", "inStock": 200 } ]
